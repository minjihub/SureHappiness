# SureHappiness
바쁜 일상 속 <소확행> 습관들이기 (2019.08 프로젝트)

## 기능
- 자신의 소확행 목표 및 횟수 설정 (ex.하루에 1번 명상하기-한달-30번)
- 설정한 목표를 시행했을 경우 도장찍기
- 목표 횟수를 완료할 경우 성공 리스트 노출(일명 '공든탑')
- 목표 달성을 많이 할수록 랭킹 상승
<br>

## 언어
- Java
- Kotlin
<br>
